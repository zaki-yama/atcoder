# ABC449 C - 解法メモ

## アプローチ

ループの `i`（= 問題の j）が右に進むたびに、`cnt` が「i から L〜R 手前の範囲」を表す窓を維持する。

サンプル1 (N=6, L=2, R=4, S=`aabcba`) で図解する。

```
index:  0   1   2   3   4   5
char:   a   a   b   c   b   a
```

---

### i=2 のとき → 窓 = [0, 0]

```
        [0]
a   a   b   c   b   a
0   1   2   3   4   5
        ^
        i=2 (j)
```

- `s[i-L] = s[0] = 'a'` を cnt に追加 → cnt[a]=1
- `cnt[s[2]] = cnt[b] = 0` → ans += 0

---

### i=3 のとき → 窓 = [0, 1]

```
      [0,1]
a   a   b   c   b   a
0   1   2   3   4   5
            ^
            i=3 (j)
```

- `s[1]='a'` を追加 → cnt[a]=2
- `cnt[s[3]] = cnt[c] = 0` → ans += 0

---

### i=4 のとき → 窓 = [0, 2]

```
         [0,2]
a   a   b   c   b   a
0   1   2   3   4   5
                ^
                i=4 (j)
```

- `s[2]='b'` を追加 → cnt[b]=1
- `cnt[s[4]] = cnt[b] = 1` → **ans += 1** (i=2, j=4 の組が見つかった)

---

### i=5 のとき → 窓 = [1, 3]

```
            [1,3]
a   a   b   c   b   a
0   1   2   3   4   5
                    ^
                    i=5 (j)
```

- `s[i-L] = s[3]='c'` を追加 → cnt[c]=1
- `s[i-R] = s[1]='a'` を**削除** → cnt[a]=1 (位置0が窓から外れる)
- `cnt[s[5]] = cnt[a] = 1` → **ans += 1** (i=1, j=5 の組が見つかった)

---

## まとめ

| ステップ | 窓の範囲 | cnt の操作 | ans |
|---------|---------|-----------|-----|
| i=2 | [0,0] | cnt[a]+=1 | 0 |
| i=3 | [0,1] | cnt[a]+=1 | 0 |
| i=4 | [0,2] | cnt[b]+=1 | 1 |
| i=5 | [1,3] | cnt[c]+=1, cnt[a]-=1 | 2 |

各ステップで：
- **窓の右端**：`s[i-L]` を追加
- **窓の左端**：`s[i-R]` を削除

これにより `cnt` が常に「距離 L〜R 以内の文字数」を保持している。
